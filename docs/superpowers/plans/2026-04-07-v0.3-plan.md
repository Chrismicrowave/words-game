## Step 3 — Settings Panel UI
**Branch:** `v0.3/settings-panel`

Overlay panel (same pattern as DailyPickerPanel — fullscreen dim + card).
- Add `SettingsBtn` to `HUD-Btns`
- Panel has left sidebar nav: **Audio | Display | Gameplay**
- Clicking nav item swaps the active sub-panel
- **Reset to Defaults** button at bottom — calls `SettingsManager.ResetToDefaults()` and refreshes all UI
- **Close** button
- Blocks gameplay input on open, restores on close

## Step 4 — Audio Settings
